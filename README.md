# BigDataFraude_ML-GraphX
Détection de fraude bancaire avec Spark MLlib et GraphX

## Technologies
- Apache Spark
- PySpark
- MLlib
- GraphX / GraphFrames
- Spark Streaming
- Grafana

## Dataset
Credit Card Fraud Detection (Kaggle)

## Pipeline
CSV → Nettoyage → SQL Analytics → ML → Graph → Streaming → Dashboard
